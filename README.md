# 🏠 Controle de Estoque Residencial

Um aplicativo web simples e eficiente para gerenciar a despensa de casa, focado em praticidade e controle de compras.

## 🚀 Funcionalidades
- **Cadastro Completo:** Nome, Marca, Medida e Categoria.
- **Gestão de Estoque:** Definição de estoque atual vs. estoque ideal.
- **Lista de Compras Automática:** A tela inicial destaca exatamente o que falta e a quantidade a comprar.
- **Modo Contagem:** Filtro por categoria para facilitar a conferência rápida dos itens.
- **Persistência de Dados:** Os dados são salvos localmente no navegador (LocalStorage).

## 📱 Como usar como um Aplicativo no Celular
1. Acesse o link gerado pelo GitHub Pages no seu navegador (Chrome ou Safari).
2. Clique no ícone de **Compartilhar** (iOS) ou nos **três pontinhos** (Android).
3. Selecione **"Adicionar à Tela de Início"**.
4. Pronto! Ele aparecerá com um ícone na sua grade de aplicativos.

## 🛠️ Tecnologias
- HTML5
- CSS3 (Variáveis modernas e Layout Responsivo)
- JavaScript Vanilla (Lógica de persistência e filtros)

## 🏗️ Como Ativar no GitHub
1. Suba o arquivo `index.html` para o seu repositório.
2. Vá em **Settings** (Configurações).
3. No menu lateral, clique em **Pages**.
4. Em "Build and deployment", escolha a branch `main` e clique em **Save**.
5. Aguarde 1 minuto e o GitHub fornecerá o link oficial do seu sistema.
